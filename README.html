General capabilities:

  • If you go to a PHP page while logged out, you will be redirected to the login page
  • If you go to the login page while logged in, you are redirected to your profile page
  • Going to a functional page results in being redirected to the 404 page
  • A user can logout, change their email or password, and navigate the visual pages

FUNTIONAL PHP

config.php

  • global variables and functions for using in other PHP files

clearNotif.php

  • Deletes a notification from the database, then redirects to the relevant post page

post_image.php

  • handles posting a text or image post

login.php

  • Handles loggin in a user
  • Verifies password using a strong hash function with salt
  • Sets the session variables for loggin in a user (username and ID)

logout.php

  • Logs out a user by destroying session data and redirecting to the login page

post_comment.php

  • Posts a comment to the database
  • Performs checks to make sure the comment isn't too short or long

signup.php

  • Registers a new user for the site
  • Hashes password, makes sure username is available, etc.
  • User is redirected back to login page to login if they are correctly registered

vote.php

  • Updates a user's vote on a post
  • If the user already voted on that post, the old vote is replaced by the new one

updateProfile.php

  • Updates the logged in user's password and/or email address
  • Both entered passwords must match

VISUAL PHP

header.php

  • The header at the top of every page
  • Shows user information when user is logged in
  • Logo is a link to homePage.php

homePage.php

  • The main page, shows a list of all posts on the site
  • Has options to sort posts by number of votes or time posted
  • Option to view cute posts, discussion posts, or all posts
  • Possible to view all posts by a given user using a GET request
  • Posts can be searched for keywords

loginPage.php

  • Users can choose to register for an account or log into Fetch
  • Users are redirected here if they are not logged in and attempt to use the site

postPage.php

  • A form for the user to make their own post
  • Users can fill in a title, text area, and upload an image to post
  • The post can be made under the cute or discussion categories

profilePage.php

  • Shows the username and number of treats for the logged in user
  • The user can change their password or email

onePostPage.php

  • Shows a single post and the comments on the post
